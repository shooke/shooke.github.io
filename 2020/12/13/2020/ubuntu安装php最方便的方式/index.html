ubuntu安装php最方便的方式

2020-12-13
笔记
  1. 添加ppa源

sudo add-apt-repository ppa:ondrej/php

  1. 升级源

sudo apt-get update && sudo apt-get upgrade

接下来就可以安装各种不同版本了

  1. sudo apt-get install php7.0 php7.0-mysql libapache2-mod-php7.0 php7.0-mcrypt php7.0-curl
