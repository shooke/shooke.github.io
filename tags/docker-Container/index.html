docker容器(Container)

03/03

容器就是独立运行起来的程序,一个容器就好比你在虚拟机里装好的一个系统。

我们可以创建、启动、停止、删除它。同时也可以将容器需要的数据放在容器之外。比如有一个数据库容器，里面有mysql服务，但数据不能放在容器里。我们可以把数据放在主机里，通过映射的方式让容器访问。
